Today Im learning about variables in javascript

variable is something like a container that holds a value, that value can be vary from string to number or it can be a array or object.

let name = Rupjyoti; here name is a variable that holds a string value.

We can declaer a variable using

  • let
  • const
  • var

let: let has a block scope,that means we cannot access a variable out side from a block {} and we can not redeclear a variables using let within the same block.i.e suppose let name = "rupjyoti" then again we can not write let name = "rohit", we can only reassign the value.
let name = "Rupjyoti";
name = "rohit";


const: Similarlly const also have block scope, the difference of let and const is

  • we can not reassign values in const
  • and every time we are using const , we have to assign a value, as like let we can not declear const without assigning value
